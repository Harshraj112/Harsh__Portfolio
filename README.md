# 🚀 Harsh Raj — Cosmic Portfolio

> **Orbiting Creativity, Anchored in Code**

A space-themed personal portfolio website showcasing my projects, skills, education, achievements, and more — built with pure HTML, CSS, and JavaScript.

🔗 **Live Site:** [harshraj-portfolio.netlify.app](https://harsh-portfolio112.netlify.app/) 

---

## ✨ Features

- 🌌 **Immersive Space Theme** — Animated star-field canvas, nebula blobs, and custom spaceship cursor
- 🎥 **Video Hero Section** — Cinematic landing with a looping space background video
- 📜 **Scroll Animations** — Smooth reveal animations powered by AOS & GSAP
- 🛠️ **Skills Arsenal** — HUD-style progress bars for AI/ML and Web Development skills
- 🎓 **Education Timeline** — Glassmorphic cards with stats and highlights
- 🏆 **Achievements Section** — Timeline view of scholarships, certifications, and leadership roles
- 🚀 **Stellar Projects** — Showcased with gradient banners, tech tags, and live demo links
- 📂 **Dedicated Project Page** — Parallax scrolling project showcase with GSAP ScrollTrigger
- 📬 **Contact Form** — Space-themed contact section with Netlify form integration
- 📱 **Fully Responsive** — Optimized for mobile, tablet, and desktop

---

## 🛠️ Tech Stack

| Layer         | Technologies                                      |
| :------------ | :------------------------------------------------ |
| **Structure** | HTML5, Semantic Elements                          |
| **Styling**   | CSS3, Custom Properties, Glassmorphism, Gradients |
| **Animation** | GSAP, ScrollTrigger, AOS, Animate.css             |
| **3D/Canvas** | Three.js, Canvas API (star-field & cursor engine) |
| **Fonts**     | Orbitron, Rajdhani, Space Mono (Google Fonts)     |
| **Icons**     | Ionicons                                          |
| **Hosting**   | Netlify                                           |
